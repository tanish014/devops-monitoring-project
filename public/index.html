🚀 DevOps & Cloud Computing Project using AWS EC2

Automated CI/CD Pipeline
using Modern DevOps Tools

This project demonstrates a complete DevOps workflow integrating GitHub, Jenkins, Docker, Terraform, AWS EC2, Prometheus, and Grafana to automate application deployment, cloud provisioning, monitoring, and observability.

GitHub

Used for source code management and version control integration.

Jenkins

Automates CI/CD pipeline stages including build and deployment.

Docker

Containerizes the application for consistent deployment across environments.

Terraform

Provisioned AWS infrastructure automatically using Infrastructure as Code.

AWS EC2

Hosts the application publicly on scalable cloud infrastructure.

Prometheus

Collects system and application monitoring metrics in real-time.

Grafana

Visualizes monitoring data through interactive dashboards and graphs.

Node Exporter

Provides CPU, memory, disk, and server-level metrics to Prometheus.
